二叉树

二叉树是一种十分常见的数据结构，本文将对二叉树进行讲解。

二叉树定义

A binary tree is a restriction where each node has exactly two children:

  • Each child is either empty or another binary tree
  • This restriction allows us to label the children as left and right subtrees

二叉树分类

perfect binary tree

定义

一个二叉树是完美二叉树，它必须满足如下的要求

  • All leaf nodes have the same depth
  • All other nodes are full

性质

常用的四条定理

  • A perfect binary tree of height $h$ has $2^{h + 1} – 1$ nodes
  • The height is $\Theta(\ln(n))$
  • There are $2^h$ leaf nodes
  • The average depth of a node is $\Theta(\ln(n))$

其中，第四条难以直接看出，这里提供一个简要的证明：
$$
\frac{\sum\limits_{k=0}^hk2^k}{2^{h+1}-1}=\frac{h2^{h+1}-2^{h+1}+2}{2^{h+1}-1}=h-1+\frac{h+1}{2^{h+1}-1}\approx h-1 = \Theta(\ln(n))
$$

complete binary tree

二叉树遍历

二叉树的遍历主要分为三种

  • 前序遍历
  • 中序遍历
  • 后序遍历

因此下文将对这三种遍历方式逐个进行讲解

前序遍历

所谓前序遍历，简单来说就是按照“根左右”的顺序遍历。

中序遍历

所谓中序遍历，简单来说就是按照“左根右”的顺序遍历。

后序遍历

所谓后序遍历，简单来说就是按照“左右根”的顺序遍历。
